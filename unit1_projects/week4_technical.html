Enumerable Methods: #cycle

Phase 0 Unit 2: Week 4


Today we will be learning about the enumerable method #cycle. This method calls block for each element of enum repeatedly n times or forever if none or nil is given. If a non-positive number is given or the collection is empty, does nothing. Returns nil if the loop has finished without getting interrupted.


Example:
a = ["a", "b", "c"]
a.cycle { |x| puts x } # print, a, b, c, a, b, c,.. forever.
a.cycle(2) { |x| puts x } # print, a, b, c, a, b, c.


Let's say you are at a family barbecue and you need to make a list for your family members so that they can submit their contact information to you for invitations and evites to your wedding. First, you would create an array of all the contact information you want your family members to provide. Then you would want to call the #cycle method on that array with the number of family members you have at the barbecue. Then copy and paste your list into a document and print it out so that you can pass it around to your family members to fill out.
